on the top of document. To make a more tidier version of data frame, we
need to skip the first row of this data. After that, I converted all the
variables’ name into lower case and use “rename()” function to rename
the variable “study_id”. Then, I reshape the provided data frame from a
wider format to a longer format.To get a tidire version of given data, I
also change the expression of the specific time node.
